Open source is not a religious cult.

Open source is not an anti-capitalist Communist plot.

Open source is not anarchist, piratical, slackerish, touchy feely, or delusional.

But open source is subversive, disruptive, empowering in ways that are changing how we work, how we communicate, how we empower ourselves.

This presentation is about all that and more — plus kittens! This is about how open source is a business decision that can yield tangible benefits. A lot of people don’t “get” open source, or aren’t quite sure they do. This session is tagged not just for beginners, but also for intermediate and advanced folks, because that doesn’t mean they’re not new to open source, the open source paradigm or the open source ecosystem.
