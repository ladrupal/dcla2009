In this session, we’ll demo Open Atrium to show how it works as a team portal, look under the hood at it’s six features, and talk about where we see it going with the addition of more features built by, well, anyone. By the end of the talk, you will know how to set up Open Atrium for your team or clients, customize it, and develop new features.

We’ll talk a lot about the Features module and about the decentralized feature model that we’re taking with Open Atrium that will make it easy for anyone to create and share a feature that plugs right into Open Atrium. The talk will also cover details on the evolution of Open Atrium and talk about how it uses and helps grow modules like notifications and casetracker and led to the creation of modules like spaces and features.
