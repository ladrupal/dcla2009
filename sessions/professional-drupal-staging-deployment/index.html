The development of large Drupal websites can pose several unique challenges when changes need to be deployed across different servers. There are as many issues to solve as there are approaches to solving them (e.g. AutoPilot, Deploy, dbscripts, and so on) and this session will include a demonstration of the deployment tools and methodology used by Drupal service firms today.

Rough agenda

  1. Defining the problem
  2. Brief overview of existing solutions
  3. Demonstration of selected tools and practices
  4. Questions and answers
